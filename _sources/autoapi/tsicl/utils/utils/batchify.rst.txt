tsicl.utils.utils.batchify
==========================

.. py:function:: tsicl.utils.utils.batchify(x: torch.Tensor | List[torch.Tensor] | None, batch_size: int, device: torch.device | None = None, batch_len: int = 0) -> Generator[torch.Tensor | None, None, None]

   Convert tensor or list into batches of desired size + move to device.

   :param x: Tensor with first dimension as batch dim, or list of tensors.
   :type x: torch.Tensor | List[torch.Tensor] | None
   :param batch_size: Batch size.
   :type batch_size: int
   :param device: Where to move the batch data to.
   :type device: torch.device, optinal
   :param batch_len: If `x` is None, yield None `batch_len` times.
   :type batch_len: int

   :returns: **out** -- Iterator over batches.
   :rtype: Generator
